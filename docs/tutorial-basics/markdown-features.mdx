# Recursos de Markdown

O Docusaurus suporta **[Markdown](https://daringfireball.net/projects/markdown/syntax)** e alguns **recursos adicionais**.

## Front Matter

Documentos Markdown possuem metadados no topo chamados de [Front Matter](https://jekyllrb.com/docs/front-matter/):

```text title="my-doc.md"
// highlight-start
---
id: my-doc-id
title: My document title
description: My document description
slug: /my-custom-url
---
// highlight-end

## Título em Markdown

Texto em Markdown com [links](./hello.md)
```

## Links

Links Markdown comuns são suportados, usando caminhos de URL ou caminhos de arquivos relativos.

```md
Vamos ver como [Criar uma página](/create-a-page).
```

```md
Vamos ver como [Criar uma página](./create-a-page.md).
```

**Resultado:** Vamos ver como [Criar uma página](./create-a-page.md).

## Imagens

Imagens comuns em Markdown são suportadas.

Você pode usar caminhos absolutos para referenciar imagens no diretório `static` (`static/img/docusaurus.png`):

```md
![Docusaurus logo](/img/docusaurus.png)
```

![Docusaurus logo](/img/docusaurus.png)

Você também pode referenciar imagens de forma relativa ao arquivo atual. Isso é útil para manter imagens próximas aos arquivos Markdown que as usam:

```md
![Docusaurus logo](./img/docusaurus.png)
```

## Blocos de código

Blocos de código em Markdown são suportados com destaque de sintaxe.

````md
```jsx title="src/components/HelloDocusaurus.js"
function HelloDocusaurus() {
  return <h1>Hello, Docusaurus!</h1>;
}
```
````

```jsx title="src/components/HelloDocusaurus.js"
function HelloDocusaurus() {
  return <h1>Hello, Docusaurus!</h1>;
}
```

## Admoestações

O Docusaurus possui uma sintaxe especial para criar admoestações e callouts:

```md
:::tip Minha dica

Use esta opção de recurso incrível

:::

:::danger Tome cuidado

Esta ação é perigosa

:::
```

:::tip Minha dica

Use esta opção de recurso incrível

:::

:::danger Tome cuidado

:::

## MDX e Componentes React

O [MDX](https://mdxjs.com/) pode tornar sua documentação mais **interativa** e permite usar **componentes React dentro do Markdown**:

```jsx
export const Highlight = ({children, color}) => (
  <span
    style={{
      backgroundColor: color,
      borderRadius: '20px',
      color: '#fff',
      padding: '10px',
      cursor: 'pointer',
    }}
    onClick={() => {
      alert(`Você clicou na cor ${color} com o rótulo ${children}`)
    }}>
    {children}
  </span>
);

Este é o <Highlight color="#25c2a0">verde do Docusaurus</Highlight>!

Este é o <Highlight color="#1877F2">azul do Facebook</Highlight>!
```
